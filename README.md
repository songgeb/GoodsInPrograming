# GoodsInPrograming

## [FindProv.sh](./FindProv.sh) (iOS)

通过可读的`Provisioning Profile`名，在本地查找相应的`mobileprovision`文件。

用法: `./FindProv.sh 'MyDevelopProvisioning'`
